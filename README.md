

###基础通信协议 
	简单的基础的自定义通信协议       
			+-------+----+-----+----------------+     
	bytes   |   4  |  1  | 3   |  ...           |
			+------+-----+-----+----------------+     
	packet  | 长度 | 编码 | 保留 | 包体           |
			+------+-----+-----+----------------+ 
 

###web分支

为区分报文类型在包体加上**4字节**，**因业务不同**，故放包体中 

				包体       
			+--------+----------------+     
	bytes   |    4   |   ...          |
			+--------+----------------+   
	packet  | 报文类型|    报文内容     |
			+--------+----------------+ 

*  实现简单的推送（android）
